  2. Edit the bookmark — replace the URL with the text in the box below.
  3. To use: open a claude.ai/code session, tap the bookmark.

Configuration (optional)

After updating, re-drag the link above or re-copy the text for iPhone.

What it does

  1. Clicks all "Show more" buttons to expand truncated messages.
  2. Extracts every message, labeling each as [USER] or [ASSISTANT].
  3. If a capture URL is configured, POSTs the transcript to your memsync daemon.
  4. Otherwise, copies to clipboard (or shows a text box on iPhone).
